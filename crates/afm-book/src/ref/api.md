# API Reference

The Rust API reference is generated by `cargo doc` and published
alongside this book at:

> [**/afm/api/**](../api/index.html)

Start at [`afm_parser`](../api/afm_parser/index.html) for the parse /
render / serialize entry points. The related crates are:

- [`afm_syntax`](../api/afm_syntax/index.html) — the `AozoraNode` AST
  types and accent-decomposition tables.
- [`afm_lexer`](../api/afm_lexer/index.html) — the 7-phase
  pure-functional Aozora recogniser.
- [`afm_encoding`](../api/afm_encoding/index.html) — Shift_JIS
  decoding and Aozora Bunko gaiji resolution.

When viewing this handbook locally (e.g. via `just book-serve`) the
API link above will 404 — run `cargo doc --workspace --no-deps` and
mount the resulting `target/doc/` at `/afm/api/` to mirror the
published Pages layout, or visit the published site at
<https://p4suta.github.io/afm/api/> directly.
