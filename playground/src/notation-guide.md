# aozora-md 記法リファレンス

aozora-md（Aozora Flavored Markdown）は **CommonMark + GFM** に **青空文庫記法** を
重ね合わせた記法です。ふつうの Markdown をそのまま書きつつ、ルビ・傍点・
縦中横などの日本語組版を `［＃…］` の注記で表現できます。

## Markdown（CommonMark）

ふつうの Markdown がそのまま使えます。

### 見出し

```
# 大見出し
## 中見出し
### 小見出し
```

### 強調とコード

```
*強調* または _強調_
**強い強調**
`インラインコード`
```

### リスト

```
- 箇条書き
- 箇条書き
  - 入れ子

1. 番号付き
2. 番号付き
```

### 引用とコードブロック

````
> 引用文

```
コードブロック
```
````

## GFM 拡張

GitHub Flavored Markdown の拡張も使えます。

### 表

```
| 左 | 中央 | 右 |
|:---|:---:|---:|
| a | b | c |
```

### 取り消し線とタスクリスト

```
~~取り消し線~~

- [x] 完了したタスク
- [ ] 未完了のタスク
```

## 青空文庫記法

日本語組版のための注記です。Markdown と混在して書けます。

### ルビ（ふりがな）

親文字を `｜`（全角縦棒）で始め、読みを `《 》` で囲みます。

```
｜青空《あおぞら》文庫
```

漢字列の直後に `《 》` を置くと、`｜` は省略できます。

```
青空《あおぞら》
```

### 傍点

```
これは［＃「ここ」に傍点］が付きます。
```

### 縦中横

縦書き時に数字などを横向きに組みます。

```
［＃縦中横］12［＃縦中横終わり］
```

### 外字

JIS にない文字を注記で示します。

```
※［＃「弓＋椀のつくり」、第3水準1-84-22］
```

### 字下げ（ブロック）

```
［＃ここから2字下げ］
段落全体が2字下げになります。
［＃ここで字下げ終わり］
```

### 見出し注記

```
［＃大見出し］第一章［＃大見出し終わり］
```

### 改ページ

```
［＃改ページ］
```

## 縦書きと横書き

ツールバーの「縦書き / 横書き」で、プレビューの組方向を切り替えられます。
ルビ・傍点・縦中横は組方向に追従します。

## エディタの便利機能

- **記法で囲む**: テキストを選択してツールバーの「囲む」から、ルビ・傍点
  などで囲めます（`Ctrl/Cmd+Alt+R` でルビ、`+B` で傍点）。
- **共有リンク**: 「共有リンクをコピー」で、今の内容を URL に埋め込んで
  共有できます。
- **配色**: ツールバーの配色ボタンで、自動 / ライト / ダークを切り替え
  られます。
