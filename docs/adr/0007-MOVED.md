# 0007. (moved)

The original ADR-0007 ("Corpus sweep strategy") records the I1–I10
invariants enforced over an arbitrary aozora-format corpus. Those
invariants are aozora-layer properties (no `［＃` leak, I3 round-trip
fixed point, …) and moved with the lexer / parser to the sibling
`aozora` repo.

It now lives at `aozora/docs/adr/0005-corpus-sweep-strategy.md`.
